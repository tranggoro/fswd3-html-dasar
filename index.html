Pengertian HTML

HTML adalah singkatan dari Hypertext Markup Language. Ia adalah bahasa markah yang dibangun dan disusun untuk merepresentasikan sebuah halaman web.
Jika sebuah halaman website adalah tubuh manusia, maka HTML ini kita bisa ibaratkan sebagai tulangnya.
Kenapa? Karena ia merepresentasikan struktur bagan tubuh sebuah halaman web
